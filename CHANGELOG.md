# Open Fit - Changelog

## [1.0.0] - Initial Release

### Added
- Complete project restructure with clean architecture
- Domain layer with entities, use cases, and repository interfaces
- Data layer with Isar models and repositories
- State management with Riverpod + code generation
- Navigation with go_router ( proper nested navigation)
- Synthwave theme preserved
- Reusable shared widgets

### Changed
- Updated UI with modern card-based layouts
- Added new screens:
  - Active workout screen
  - Food search screen
  - Glucose entry screen
  - Profile screen

### Features
- **Workouts**: Start/stop, templates, exercise library, history,- **Nutrition**: Macro tracking, meal logging, water intake, barcode scanning
- **Health**: Glucose logging, A1C estimation, insulin tracking (coming)
- **Metrics**: Weight tracking, body measurements, progress photos

### Known Issues
- None yet! This is a a clean slate.
